Para executar o programa principal ou de teste
é necessário seguir os seguintes passos, através
do termina do linux:

- navegar até a pasta do projeto
- dentro da pasta do projeto executar os seguintes comandos:
  1 "sudo chmod +x [main || test]"
  2 "sudo ./[main || test]"

Ps.: use o sudo somente se necessário

A classe unitTest.cpp tem vários exemplos dos construtores
e métodos de cada classe.

O slide de apresentação e o diagrama UML das classes se encontram
na pasta docs
